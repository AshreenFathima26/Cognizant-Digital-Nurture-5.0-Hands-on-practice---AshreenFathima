# 🚀 Week 4 – Spring Cloud Microservices & SonarQube

## 📌 Overview

This repository contains the hands-on exercises completed during **Week 4** of the **Cognizant Digital Nurture 5.0 Deep Skilling Program**.

The primary objective of this week was to understand enterprise-level application development using **Spring Cloud Microservices** and to improve software quality through **SonarQube Static Code Analysis**.

The implementation demonstrates how independent microservices communicate through a service registry and API Gateway while following modern software development practices.

---

# 🎯 Learning Objectives

- Understand Microservices Architecture
- Implement Service Discovery using Eureka Server
- Configure API Gateway for request routing
- Develop independent Spring Boot Microservices
- Learn Service Registration and Discovery
- Perform Static Code Analysis using SonarQube
- Improve Code Maintainability and Quality
- Follow Enterprise Java Development Practices
