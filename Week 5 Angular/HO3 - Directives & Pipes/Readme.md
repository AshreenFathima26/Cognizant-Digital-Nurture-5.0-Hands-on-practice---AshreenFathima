# 📚 HO3 – Directives & Pipes

## Cognizant Digital Nurture 5.0 – Java FSE (Angular)

### Student Course Portal

This Hands-on demonstrates the implementation of Angular Directives and Pipes by extending the Student Course Portal developed in previous hands-ons. The application provides an interactive interface for viewing and managing available courses while showcasing Angular's built-in and custom features.

---

## 📌 Learning Objectives

- Understand Angular Structural Directives
- Implement Attribute Directives
- Work with Built-in Pipes
- Create and use a Custom Pipe
- Enhance UI using Angular features
