# 📝 HO4 – Template-Driven Forms & Validation

## Cognizant Digital Nurture 5.0 – Java FSE (Angular)

### Student Course Portal – Enrollment Request Form

This Hands-on demonstrates the implementation of **Template-Driven Forms** in Angular using `ngForm` and `ngModel`. The Student Course Portal is extended with an Enrollment Request page that includes built-in validation, error handling, form submission, and reset functionality.

---

## 📌 Learning Objectives

- Understand Template-Driven Forms
- Implement Two-Way Data Binding using `ngModel`
- Work with `ngForm`
- Apply Built-in Validators
- Display Validation Error Messages
- Handle Form Submission
- Improve User Experience with Form Validation
