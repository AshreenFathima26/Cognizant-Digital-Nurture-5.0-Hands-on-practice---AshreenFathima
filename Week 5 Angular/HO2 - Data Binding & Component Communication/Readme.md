# HO2 – Data Binding & Component Communication

## Cognizant Digital Nurture 5.0
### Week 5 – Angular Frontend Development

## Objective
Implement Angular Data Binding and Component Communication concepts by developing a Student Course Portal.

## Concepts Covered

- Interpolation
- Property Binding
- Event Binding
- Two-Way Data Binding
- Parent to Child Communication (@Input)
- Child to Parent Communication (@Output)
- EventEmitter
- Angular Component Lifecycle
- Dynamic Search using ngModel
- Navigation using Angular Routing

## Project

**Student Course Portal**

### Features

- Professional Home Page
- Dynamic Banner Image
- Search Courses using Two-Way Data Binding
- Enroll Button using Event Binding
- Course Cards
- Parent–Child Component Communication
- Live Search Text Display
- Student Statistics Dashboard
- Angular Routing (Home, Courses, Profile)

## Technologies Used

- Angular 19
- TypeScript
- HTML5
- CSS3
