# 🎓 Student Course Portal

## Cognizant Digital Nurture 5.0 - Week 5 Angular

---

## 📌 Project Overview

Student Course Portal is an Angular application developed as part of the Cognizant Digital Nurture 5.0 Deep Skilling Program.

This project demonstrates modern Angular concepts including routing, nested routing, query parameters, route guards, reactive forms, and component-based architecture.

---

## 🚀 Features

### 🏠 Home Page
- Student Course Portal Dashboard
- Navigation Header

### 📚 Course Management
- Display available courses
- Search courses
- View course details
- Course cards
- Enroll / Unenroll functionality

### 🎯 Routing
- Angular Routing
- Nested Routing
- Dynamic Route Parameters
- Query Parameters
- Wildcard (404) Route

### 🔒 Route Guards
- CanActivate Guard
- CanDeactivate Guard

### 📄 Forms
- Template Driven Form
- Reactive Form
- Custom Validation
- Async Validation
- Dynamic FormArray

### 👤 Student Profile
- Student Information
- Current Semester
- CGPA
- Course Summary
